## API Overview
[Restful Booker](https://restful-booker.herokuapp.com) is a public demo API for practicing API testing. Supports:
- Booking CRUD operations
- Token-based authentication
- Query parameters and error scenarios

## Test Coverage
- **40+ automated tests** in Postman
- 100% coverage of core endpoints
- Positive/Negative scenarios
- Authentication validation
- Response validation (status, body, headers)

## Postman Collection
Import `postman/RestfulBooker-Tests.postman_collection.json` into Postman.

## Setup Instructions
1. Fork/Import Postman collection
2. Set environment: `base_url = https://restful-booker.herokuapp.com`
3. Generate auth token via `/auth`
4. Run collection runner
